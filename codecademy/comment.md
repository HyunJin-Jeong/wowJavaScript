---
프로그래밍은 대부분 협동이 필요해요. 그리고 우리가 코딩을 할 때, 바라보는 코드들은 금방 이해하기 어려울 수 있습니다. 이러한 이유로, 코드에 메모를 남기는 것이 우리 자신과 또 다른 개발자들에게 유용합니다.
<br>
<br>

우리는 Javascript 코드를 작성할 때, 코멘트를 남길 수 있어요. 이 코멘트는 컴퓨터에서 Javascript 프로그램을 실행할 때, 무시합니다. 오로지 사람들이 읽기 위한 코멘트예요.
<br>
<br>

코멘트를 통해 개코드가 어떤 동작을 하는지 개발자들을 위해서 설명할 수 있고, 다른 의미로 사용되는 주석일 수도 있답니다.
<br>
<br>

---
우린 코멘트를 달 때, 두 가지 타입을 사용할 수 있어요.
<br>
<br>

첫 번째로, `//`를 이용해 한 줄 코멘트를 남기는 방법
```javascript
// Prints 5 to the console
console.log(5);
```
아래 처럼 사용할 수도 있답니다!
```javsacript
console.log(5); // Prints 5
```
<br>

두 번째로, `/* */`를 사용해 여러 줄 코멘트를 남기는 방법
```javascript
/*
This is all commented
console.log(5);
console.log(99);
*/
```
